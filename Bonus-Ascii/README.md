# ASCII Live - Harvester

Animated ASCII art of the classic Harvester game logo for your terminal.

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

Or run directly:

```bash
node index.js
```

Press `Ctrl+C` to exit the animation.

## Features

- Smooth dripping blood animation
- Red color effects using ANSI codes
- Continuous looping animation
- Works in any terminal that supports ANSI colors

## About

Inspired by the classic 1996 horror game "Harvester" and the ascii.live project.
